## Installation

MMLogWindow is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MMLogWindow', :git => 'https://github.com/MatthewMerritt/MMLogWindow.git'
```
